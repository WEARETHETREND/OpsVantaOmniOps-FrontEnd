# Access Policy

**© 2026 OpsVanta LLC — All Rights Reserved.**

This repository is restricted to authorized personnel, approved contractors,
and explicitly approved partners operating under a signed NDA.

---

## Access Rules

- **Least-privilege access only** — access is scoped to what is required for the role
- **Business-justified access** — every access request must have a documented business reason
- **No credential sharing** — repository credentials and tokens may not be shared
- **Local copy protection** — any local copies of this repository must be secured appropriately
- **Immediate offboarding** — access must be revoked immediately upon departure or contract end

---

## Prohibited Actions

Authorized users may not:

- Copy or export source code, designs, or documentation to unauthorized systems
- Redistribute any portion of this repository externally
- Discuss private implementation details in public forums
- Upload source code or internal documentation to third-party tools without written approval
- Commit secrets, credentials, or environment files to version control
