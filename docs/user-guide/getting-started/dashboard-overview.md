# Dashboard Overview

Complete guide to understanding and using your OpsVanta dashboard.

## Dashboard Layout

### Main Navigation

**Left Sidebar:**
- 🏠 Home - Dashboard overview
- 📁 Projects - Your projects list
- ➕ New Project - Create new project
- 🔧 Workflows - Automation workflows
- 📊 Analytics - Performance metrics
- ⚙️ Settings - Account settings

**Top Bar:**
- 🔍 Search - Find projects quickly
- 🔔 Notifications - Updates and alerts
- 👤 Profile - Account menu
- ❓ Help - Support resources

### Dashboard Sections

#### Projects Overview
- Recent projects grid
- Quick actions (edit, preview, publish)
- Project status indicators
- Creation date and last modified

#### Quick Stats
- Total projects
- Published websites
- Total visitors this month
- Storage used

#### Recent Activity
- Latest edits and changes
- Team member activities
- Publication history
- System notifications

## Navigation Tips

**Keyboard Shortcuts:**
- `Ctrl/Cmd + K` - Quick search
- `Ctrl/Cmd + N` - New project
- `Ctrl/Cmd + ,` - Settings
- `Ctrl/Cmd + /` - Help

**Quick Access:**
- Pin favorite projects
- Use search for instant access
- Create project templates
- Organize with folders

## Customization

**Dashboard Settings:**
- Choose default view (grid/list)
- Set items per page
- Configure notifications
- Customize sidebar

**Theme:**
- Light mode
- Dark mode
- Auto (system preference)
