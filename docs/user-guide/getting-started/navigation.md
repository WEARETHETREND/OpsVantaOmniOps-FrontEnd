# Navigating OpsVanta

Learn how to efficiently navigate the OpsVanta platform.

## Main Navigation

### Dashboard
Your home base for all activities.

**Access:** Click "Dashboard" or press `Ctrl/Cmd + H`

**Features:**
- Project overview
- Quick stats
- Recent activity
- Quick actions

### Projects
Manage all your projects.

**Access:** Click "Projects" in sidebar or press `Ctrl/Cmd + P`

**Views:**
- Grid view - Visual project cards
- List view - Detailed project list
- Filter by status, type, date

### New Project
Create new projects quickly.

**Access:** 
- Click "+ New Project" button
- Press `Ctrl/Cmd + N`
- Click "New" in top bar

### Workflows
Automate your processes.

**Access:** Click "Workflows" in sidebar

**Features:**
- View all workflows
- Create new workflows
- Monitor workflow runs
- Debug and test

### Analytics
Track performance.

**Access:** Click "Analytics" in sidebar

**Metrics:**
- Visitor statistics
- Performance data
- Traffic sources
- Conversion tracking

### Settings
Configure your account.

**Access:** 
- Click profile avatar → Settings
- Press `Ctrl/Cmd + ,`

**Sections:**
- Profile
- Security
- Billing
- Team
- Notifications
- Preferences

**Bottom Bar:**

### Keyboard Shortcuts

**General:**
- `Ctrl/Cmd + S` - Save
- `Ctrl/Cmd + Z` - Undo
- `Ctrl/Cmd + Shift + Z` - Redo
- `Ctrl/Cmd + D` - Duplicate
- `Delete` - Delete selection

**Navigation:**
- `Ctrl/Cmd + 1` - Desktop view
- `Ctrl/Cmd + 2` - Tablet view
- `Ctrl/Cmd + 3` - Mobile view
- `Ctrl/Cmd + 0` - Fit to screen
- `Ctrl/Cmd + +/-` - Zoom in/out

**Editing:**
- `Ctrl/Cmd + C` - Copy
- `Ctrl/Cmd + V` - Paste
- `Ctrl/Cmd + X` - Cut
- `Ctrl/Cmd + A` - Select all
- `Esc` - Deselect

## Search

### Quick Search
Press `Ctrl/Cmd + K` to open quick search.

**Search for:**
- Projects by name
- Pages within projects
- Components
- Settings
- Help articles

### Advanced Filters

**Filter projects by:**
- Status (draft, published)
- Type (landing page, website, etc.)
- Date created
- Last modified
- Tags

## Breadcrumbs

Follow breadcrumb trail to navigate hierarchy:
```
Dashboard > Projects > My Website > Homepage
```

Click any level to jump back.

## Tips for Efficient Navigation

1. **Learn shortcuts** - Master keyboard shortcuts
2. **Use quick search** - Fastest way to find anything
3. **Pin favorites** - Quick access to frequent projects
4. **Organize** - Use tags and folders
5. **Browser tabs** - Open multiple projects
6. **Bookmarks** - Bookmark frequently used pages

## Troubleshooting

**Navigation not working?**
- Refresh the page (F5)
- Clear browser cache
- Try different browser
- Check internet connection

**Can't find something?**
- Use search (Ctrl/Cmd + K)
- Check all project tabs
- Review recent activity
- Contact support
