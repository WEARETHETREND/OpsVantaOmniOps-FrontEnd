# Building Your First Website with OpsVanta

A step-by-step guide to creating your first website using AI.

## Overview

This tutorial will walk you through the entire process of creating a professional website with OpsVanta, from concept to publication.

**Time Required:** 15-20 minutes  
**Difficulty:** Beginner  
**Prerequisites:** OpsVanta account

## Step 1: Plan Your Website

### Define Your Purpose

Ask yourself:
- What is the main goal of this website?
- Who is my target audience?
- What actions do I want visitors to take?

### Common Website Types

| Type | Best For | Key Features |
|------|----------|--------------|
| **Landing Page** | Product launches, campaigns | Single focus, strong CTA |
| **Business Website** | Companies, services | Multiple pages, contact forms |
| **Portfolio** | Creatives, professionals | Project showcases, galleries |
| **Blog** | Content creators, writers | Article listings, categories |
| **E-commerce** | Online stores | Product catalogs, checkout |

### Gather Your Content

Prepare:
- ✅ Business/project name
- ✅ Tagline or description
- ✅ Key selling points (3-5 items)
- ✅ Contact information
- ✅ Logo (if available)
- ✅ Images or photos
- ✅ Color preferences

## Step 2: Create New Project

1. **Access Dashboard**
   - Log in to your OpsVanta account
   - You'll see your dashboard

2. **Start New Project**
   - Click the **"New Project"** button
   - Or press `Ctrl/Cmd + N` keyboard shortcut

3. **Choose Project Type**
   - Select from available templates
   - Or choose "Blank" to start from scratch

## Step 3: Use AI Generation

### Writing Your Prompt

The quality of your AI prompt directly affects the result. Here's a proven formula:

```
[Business Type] + [Purpose] + [Key Features] + [Style Preferences]
```

### Example Prompts

**Landing Page:**
```
Create a modern landing page for a fitness coaching service.
Include hero section with sign-up form, benefits list,
testimonials, pricing table, and contact section.
Use energetic colors (blues and oranges). Target audience
is busy professionals looking to get in shape.
```

**Business Website:**
```
Build a professional website for a boutique law firm
specializing in family law. Need homepage, about us,
services, attorney bios, blog, and contact page.
Use sophisticated navy blue and gold colors.
Professional but approachable tone.
```

**Portfolio:**
```
Design a creative portfolio for a graphic designer.
Showcase 8-10 projects with before/after images,
about me section, skills list, and contact form.
Minimalist black and white design with pops of color.
```

### Best Practices

✅ **Do:**
- Be specific about sections needed
- Mention target audience
- Specify color preferences
- Include must-have features
- Describe desired tone/style

❌ **Don't:**
- Use vague descriptions
- Omit important requirements
- Contradict yourself
- Write overly long prompts (keep under 200 words)

### Generate Your Website

1. **Enter Your Prompt**
   - Type or paste your prompt into the text area
   - Review for clarity and completeness

2. **Set Generation Options**
   - Choose color scheme (or let AI decide)
   - Select font pairing
   - Pick design style (modern, classic, minimal, bold)

3. **Click "Generate with AI"**
   - AI will create your website in 30-60 seconds
   - Watch the progress indicator
   - You can cancel anytime

4. **Review Generated Design**
   - AI will show you the complete website
   - Preview each section
   - Check mobile responsiveness

## Step 4: Customize Your Website

### Visual Editor

Use the drag-and-drop editor to make changes:

#### Edit Text Content

1. Click any text element
2. Type to replace content
3. Use formatting toolbar:
   - Bold, italic, underline
   - Font size and color
   - Alignment
   - Links

#### Modify Sections

- **Add Section:** Click "+ Add Section" between existing sections
- **Reorder Sections:** Drag sections up or down
- **Delete Section:** Click the trash icon
- **Duplicate Section:** Click the copy icon

#### Customize Images

1. Click any image
2. Options appear:
   - **Replace:** Upload your own image
   - **Adjust:** Crop, resize, or apply filters
   - **Position:** Change focal point
   - **Alt Text:** Add for accessibility

#### Adjust Colors

1. Open the color picker (palette icon)
2. Choose from generated palette
3. Or create custom colors
4. Apply globally or to specific elements

#### Change Fonts

1. Open font selector
2. Browse font pairings
3. Preview in real-time
4. Apply changes

### Code Editor (Optional)

For advanced users:

1. Click **"Code View"** toggle
2. Edit HTML directly
3. Add custom CSS
4. Insert JavaScript
5. Changes update in real-time

See [Code Editor Guide](../editor/code-editor.md) for details.

## Step 6: Add Pages (Optional)

### Create Additional Pages

1. Click **"Pages"** in sidebar
2. Click **"Add New Page"**
3. Choose page template or blank
4. Name your page (e.g., "About", "Contact", "Services")

### Link Pages Together

1. Select text or button to link
2. Click link icon in toolbar
3. Choose destination page
4. Set link to open in same or new window

### Create Navigation Menu

Navigation is automatically generated from your pages.

**Customize:**
1. Go to **Settings → Navigation**
2. Reorder menu items (drag and drop)
3. Show/hide pages in menu
4. Add external links

## Step 7: Publish Your Website

### Pre-Launch Checklist

✅ Content proofread and accurate  
✅ All images have alt text  
✅ Links tested and working  
✅ Mobile version looks good  
✅ Contact information correct  
✅ SEO basics configured  

### Publishing Process

1. **Click "Publish" Button**
   - Top right corner of editor
   - Green button

2. **Choose Publishing Options**
   - **Free subdomain:** `yourproject.opsvanta.com`
   - **Custom domain:** Connect your own domain
   
3. **Configure Settings**
   - Site title and description
   - Favicon
   - Social sharing image
   - Analytics tracking

4. **Confirm Publication**
   - Review summary
   - Click **"Publish Now"**
   - Your site goes live instantly!

### Post-Publication

Your website is now live! You'll get:

- ✅ Published URL
- ✅ QR code for sharing
- ✅ Social sharing links
- ✅ Embed codes

### Making Updates

After publication:

1. Edit your project anytime
2. Changes are saved as drafts
3. Click **"Publish Changes"** to update live site
4. Updates are instant

## Troubleshooting

### AI Generation Issues

**Problem:** AI didn't generate what I wanted  
**Solution:** Refine your prompt with more specific details and try again

**Problem:** Generation is taking too long  
**Solution:** Check your internet connection, or try during off-peak hours

### Publishing Problems

**Problem:** Publish button is disabled  
**Solution:** Ensure all required fields are filled (site title, description)

**Problem:** Custom domain not working  
**Solution:** See [DNS Configuration Guide](../domain-publishing/dns-configuration.md)
