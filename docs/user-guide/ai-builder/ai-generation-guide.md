## Writing Effective Prompts

### The Perfect Prompt Formula

```
[1. Project Type] + [2. Purpose] + [3. Target Audience] +
[4. Key Sections] + [5. Design Style] + [6. Special Requirements]
```

### Component Breakdown

#### 1. Project Type

Be specific:
- Landing page
- Multi-page website
- Portfolio
- Blog
- E-commerce site
- Documentation site
- Community platform

#### 2. Purpose

What should the website achieve?
- Generate leads
- Sell products
- Showcase work
- Share information
- Build community
- Provide support

#### 3. Target Audience

Who are you building for?
- Demographics (age, location, profession)
- Tech-savviness level
- Pain points they have
- What they're looking for

#### 4. Key Sections

List required sections:
- Hero/header
- Features or services
- About/story
- Testimonials/social proof
- Pricing
- FAQ
- Contact/CTA
- Footer

#### 5. Design Style

Describe visual preferences:
- Modern, classic, minimalist, bold
- Industry-appropriate aesthetics
- Color preferences
- Mood/feeling
- Reference examples (optional)

#### 6. Special Requirements

Any unique needs:
- Specific features
- Integration requirements
- Accessibility needs
- Branding guidelines
- Technical constraints

## Prompt Examples by Industry

### Technology/SaaS

```
Create a modern SaaS landing page for a project management tool
targeting remote teams. Include hero section with product demo video,
key features grid (6 features), testimonials from tech companies,
pricing table (3 tiers), integration logos, and free trial signup.
Use professional blue and white color scheme. Tech-forward but
approachable design.
```

### E-commerce/Retail

```
Design an e-commerce homepage for an organic skincare brand.
Target audience is health-conscious women 25-45. Need hero with
bestseller carousel, product categories grid, brand story section,
customer reviews, Instagram feed integration, and newsletter signup.
Use natural earthy tones (greens, beiges). Clean, modern aesthetic
with emphasis on product photography.
```

### Professional Services

```
Build a website for a boutique accounting firm serving small
businesses. Need homepage, services page, about team, client
testimonials, resources/blog, and contact form. Professional but
approachable tone. Use trustworthy colors (navy blue, white, hints
of gold). Include credibility markers (certifications, years in
business, client logos).
```

### Creative/Portfolio

```
Create a portfolio site for a freelance UX designer. Showcase 8-10
case studies with project cards (hover effects), detailed project
pages, about me section with timeline, skills visualization, client
testimonials, blog for design thoughts, and contact form. Modern,
minimalist black and white with one accent color. Typography-focused
design.

### Restaurant/Food Service

Design a website for a farm-to-table restaurant in Portland. Need
homepage with atmosphere photos, menu with filtering by dietary
restrictions, about our story, meet the chef, reservations booking,
events calendar, and contact with map. Warm, inviting design using
earth tones. Emphasis on food photography and sustainable practices.

### Real Estate

## Advanced Prompt Techniques

### Iterative Refinement

Start broad, then refine:

**First Attempt:**
```
Create a website for a yoga studio.
```

**Better:**
```
Create a website for a yoga studio targeting busy professionals.
Include class schedule, instructor bios, pricing, and online booking.
Use calming colors (soft blues, greens). Peaceful, zen aesthetic.
```

**Best:**
```
Create a website for an urban yoga studio targeting busy professionals
seeking stress relief. Include: hero with next class countdown,
interactive class schedule with filtering (style, time, instructor),
instructor profiles with teaching philosophy, membership pricing with
comparison, online booking integration, beginner's guide to yoga,
student testimonials, and contact with studio tour booking. Use
calming palette (soft sage green, warm beige, white) with natural
textures. Peaceful yet energizing design. Emphasize accessibility
for beginners and community atmosphere.
```

### Using Style References

Reference well-known brands or websites:

```
Landing page for productivity app. Design style similar to Notion
(clean, spacious, illustration-based) but with warmer colors.
Include: animated hero, feature showcase, use cases, testimonials,
and CTA for free trial.
```

### Specifying Interactions

Describe desired user interactions:

```
Portfolio with scroll-triggered animations. Projects should fade in
as user scrolls. Include smooth parallax effects on hero section.
Case study pages should have side-by-side before/after comparisons
with slider control. Contact form should have real-time validation.
```

### Mood and Tone

Use descriptive language:

```
High-energy fitness coaching site. Bold, motivating design that
makes users feel empowered. Use intense colors (bright orange,
deep blacks), strong typography, action-oriented language, and
progress-focused imagery. Should feel like a personal trainer's
pep talk.
```

## Generation Options

### Color Schemes

**Monochromatic:**
- Single color with variations
- Clean, professional
- Easy on the eyes

**Complementary:**
- Opposite colors on color wheel
- High contrast, energetic
- Good for calls-to-action

**Analogous:**
- Adjacent colors on color wheel
- Harmonious, calming

**Triadic:**

## AI Generation Tips & Tricks

### Pro Tips

💡 **Use industry-specific language** - AI understands domain terminology  
💡 **Mention competitors** - "Like X but with Y"  
💡 **Specify content types** - "With video backgrounds" or "Using illustrations"  
💡 **Request specific features** - "Interactive pricing calculator"  
💡 **Set tone explicitly** - "Professional yet playful"  
💡 **Include constraints** - "No animations" or "Must be accessible"  
💡 **Mention technical needs** - "Fast-loading" or "SEO-optimized"  

### Common Mistakes to Avoid

❌ **Too vague** - "Make a nice website"  
❌ **Too long** - 500-word prompts overwhelm the AI  
❌ **Contradictory** - "Modern and vintage" without clarification  
❌ **Unrealistic** - "Include 50 different sections"  
❌ **Missing purpose** - Not explaining what the site should achieve  
❌ **No audience** - Not specifying who it's for  

### Time-Saving Shortcuts

⚡ **Save successful prompts** - Reuse and adapt for similar projects  
⚡ **Use templates** - Start with pre-built prompt templates  
⚡ **Build a swipe file** - Collect good prompt examples  
⚡ **Create prompt library** - Organize by industry/type  

## Troubleshooting Generation

### "AI isn't understanding my prompt"

**Solution:**
- Simplify language
- Break into shorter sentences
- Remove jargon or explain it
- Use concrete examples

### "Generated site is too generic"

**Solution:**
- Add more unique details
- Specify your differentiators
- Include brand personality traits
- Mention specific content types

### "Wrong design style"

**Solution:**
- Use explicit style keywords
- Reference specific design movements
- Mention well-known brand styles
- Specify what to avoid

### "Missing sections"

**Solution:**
- List all sections explicitly
- Number them for clarity
- Specify order/priority
- Mention must-have vs. nice-to-have

### "Generation failed"

**Solution:**
- Check internet connection
- Try shorter prompt
- Remove special characters
- Simplify complex requirements
- Contact support if persists

## Case Studies

### Case Study 1: SaaS Landing Page

**Initial Prompt (Poor):**
```

```

**Result:** Generic, unclear purpose, missing features

**Refined Prompt (Good):**
```
Create a SaaS landing page for team communication software targeting
remote companies. Include hero with product screenshots, key features

free trial signup. Modern tech aesthetic with blue primary color.

**Result:** Professional, targeted, complete

### Case Study 2: Restaurant Website

**Challenge:** Convey atmosphere and menu

**Successful Prompt:**
```
Design a website for an upscale Italian restaurant. Hero with ambient
restaurant photos, reservation booking CTA, menu organized by course
(antipasti, primi, secondi, dolci) with dietary icons, chef's story
with kitchen photos, wine list, private dining info, and location with
map. Elegant design using deep burgundy, cream, and gold. Should feel
warm and inviting, emphasizing authentic Italian experience.
```

**Key Success Factors:**
- Specific menu structure
- Atmosphere description
- Clear color palette
- Emotional goal stated
