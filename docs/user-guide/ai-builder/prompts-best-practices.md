# AI Prompts Best Practices

Master the art of writing effective AI prompts to generate better websites.

## Quick Tips

✅ **Be specific** - More details = better results
✅ **Mention audience** - Who is this website for?
✅ **List sections** - What pages/sections do you need?
✅ **Specify style** - Modern, classic, minimal, bold?
✅ **Include colors** - Any color preferences?
✅ **State purpose** - What should the website achieve?

## The Perfect Prompt Formula

```
[Type] + [Purpose] + [Audience] + [Sections] + [Style] + [Colors]
```

### Example

**Good Prompt:**
```
Create a modern landing page for a meditation app targeting busy
professionals. Include hero with app screenshot, key features
(guided meditation, sleep sounds, progress tracking), testimonials,
pricing table (free, pro, teams), and email signup. Use calming
blues and purples with minimalist design.
```

**Why it works:**
- ✅ Specific type (landing page)
- ✅ Clear purpose (meditation app)
- ✅ Defined audience (busy professionals)
- ✅ Listed sections (hero, features, testimonials, pricing, signup)
- ✅ Style preference (modern, minimalist)
- ✅ Color scheme (blues, purples, calming)

## Common Mistakes

### Too Vague

❌ **Bad:**
```
Make a website for my business
```

✅ **Good:**
```
Create a professional website for an accounting firm specializing
in small business taxes. Include homepage with services overview,
about our team section, tax services page with pricing, client
testimonials, blog for tax tips, and contact form. Use trustworthy
navy blue and white color scheme with professional but approachable
design.
```

### Missing Key Information

❌ **Bad:**
```
I need a portfolio website with my work
```

✅ **Good:**
```
Build a portfolio website for a freelance graphic designer. Feature
8-10 project case studies with before/after images, about me section
with design philosophy, services offered (branding, web design, print),
client testimonials, and contact form. Use bold, creative design with
black, white, and one vibrant accent color.
```

### Contradictory Requirements

❌ **Bad:**
```
Create a website that's both vintage and futuristic, very simple
but also feature-rich, professional yet playful
```

✅ **Good:**
```
Create a website with modern retro-inspired design (think 80s neon
signs with clean modern layout). Professional tone with occasional
playful elements. Balance feature showcase with clean, uncluttered
design.
```

## Prompt Templates

### Landing Page Template

```
Create a [style] landing page for [product/service] targeting
[audience]. Include:
- Hero section with [call-to-action]
- [Number] key features or benefits
- [Social proof type: testimonials/logos/stats]
- Pricing section with [number] tiers
- [Other sections]

Use [color preferences] color scheme. [Additional style notes].
```

### Business Website Template

```
Build a [number]-page website for [business type] serving [target
market]. Need:
- Homepage with [key elements]
- [List other pages: About, Services, Contact, etc.]
- [Special features: booking, forms, etc.]

Style: [modern/classic/minimal/etc]
Colors: [color preferences]
Tone: [professional/friendly/etc]
```

### Portfolio Template

```
Design a portfolio website for [profession] specializing in [niche].
Showcase:
- [Number] projects/works with [description style]
- About section highlighting [key points]
- [Additional sections]

Visual style: [description]
Layout: [grid/masonry/full-screen/etc]
Colors: [preferences]
```

## Industry-Specific Examples

### SaaS Product

```
Create a modern SaaS landing page for a team collaboration tool
competing with Slack. Target: remote teams at tech startups.

Sections needed:
- Hero: Product demo video, "Start Free Trial" CTA
- Features: Real-time messaging, file sharing, integrations, video calls
- Use cases: Engineering teams, marketing teams, customer support
- Customer logos: Show 20+ recognizable brands
- Testimonials: 3-4 with photos and titles
- Pricing: Free, Pro ($12/user/month), Business ($25/user/month)
- FAQ: Common questions about security, integrations, migration
- Footer: Full sitemap, social links, security badges

Design: Modern, tech-forward, inspired by Linear and Notion
Colors: Primary blue (#4F46E5), clean whites and grays
Vibe: Professional yet approachable, innovative
Special requests: Dark mode friendly, lots of white space
```

### Restaurant

```
Design a website for an upscale Italian restaurant in downtown Chicago.
Target: Locals and tourists seeking authentic Italian dining.

Pages:
- Homepage: Beautiful food photography, location/hours, reservation CTA
- Menu: Organized by course (Antipasti, Primi, Secondi, Dolci)
  with descriptions and prices
- About: Chef's story, imported ingredients, 20 years in Chicago
- Private Dining: Info for events, capacity, sample menus
- Gallery: Professional photos of dishes, ambiance, events
- Reviews: Aggregate rating, highlighted reviews
- Contact: Reservation system, map, parking info, contact form

Design: Elegant, warm, inviting
Colors: Deep burgundy, cream, gold accents
Typography: Serif headings, clean sans-serif body
Mood: Upscale but welcoming, emphasizing tradition and quality
```

### Freelance Designer

```
Build a portfolio for a UX/UI designer with 5 years experience.
Focus: Mobile app design for consumer products.

Structure:
- Homepage: Hero with latest project, brief intro, quick stats
- Work: 8 case studies in grid layout
  * Each project shows: thumbnail, title, company, role, year
  * Click through to full case study with:
    - Challenge, process, solution
    - Multiple screenshots/mockups
    - Results/metrics if available
- About: Photo, story, design philosophy, current availability
- Process: How I work, typical timeline, deliverables
- Contact: Email, LinkedIn, Dribbble, calendar for consultations

Design: Clean, minimalist, let work shine
Colors: Black, white, single accent color (blue)
Layout: Generous white space, large images
Typography: Modern sans-serif, excellent hierarchy
Interactive: Smooth transitions, hover effects on projects
```

## Advanced Techniques

### Using References

```
Create a landing page similar to Stripe's website (clean, modern,
excellent typography, subtle animations) but for a fintech app
targeting Gen Z users. Make it more playful and colorful while
maintaining professionalism.
```

### Specifying Interactions

```
Homepage with scroll-triggered animations. As user scrolls:
- Features fade in from bottom
- Stats counter animates from 0
- Testimonials slide in from sides
- Parallax effect on hero image
- Sticky navigation after scroll
```

### Mobile-First Requirements

```
Design mobile-first landing page for fitness app. Mobile version must:
- Load in under 2 seconds
- Feature thumb-friendly buttons
- Minimize scrolling with collapsible sections
- Prioritize "Download App" CTA above fold
- Show app screenshots in mobile frame
```

## Testing Your Prompts

### Before Generating

Ask yourself:
1. Is my website type clear?
2. Did I specify my audience?
3. Are all needed sections listed?
4. Is the style clearly described?
5. Are colors mentioned?
6. Is the purpose obvious?

### After Generating

If results aren't perfect:
1. Be more specific
2. Add missing details
3. Reference examples
4. Clarify contradictions
5. Try iterative refinement

## Prompt Optimization

### Version 1 (Basic)

```
Create a website for a yoga studio
```

### Version 2 (Better)

```
Create a website for a yoga studio with class schedules,
instructor bios, and online booking
```

### Version 3 (Best)

```
Create a calming, zen-inspired website for an urban yoga studio
targeting busy professionals. Include:
- Hero with next class countdown
- Interactive class schedule (filter by style, time, instructor)
- Instructor profiles with teaching philosophy
- Membership pricing (Drop-in, Monthly, Annual)
- Beginner's guide to yoga
- Student testimonials
- Location with directions

Colors: Soft sage green (#A8B5A0), warm beige, white
Style: Peaceful, spacious, natural textures
Tone: Welcoming, accessible, emphasizing stress relief
```
