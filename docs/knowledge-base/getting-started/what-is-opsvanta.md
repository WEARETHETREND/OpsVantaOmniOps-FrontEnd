# What is OpsVanta?

## Quick Answer
OpsVanta is an AI-powered website builder that lets you create professional websites in seconds by simply describing what you want in natural language.

## Detailed Explanation

### Overview
OpsVanta combines cutting-edge artificial intelligence, modern web technologies, and intuitive design tools to make website creation accessible to everyone - from complete beginners to experienced developers.

### How It Works

**1. Describe What You Want**
Simply type or speak what kind of website you need. For example:
- "A landing page for my yoga studio with class schedules and booking"
- "A portfolio website to showcase my photography"
- "An e-commerce site for my handmade jewelry business"

**2. AI Generates Your Website**
Our AI understands your requirements and creates a complete website in 30-60 seconds, including:
- Professional layout and structure
- Matching color schemes
- Appropriate typography
- Mobile-responsive design
- Optimized performance

**3. Customize to Perfection**
Use our visual editor to:
- Edit any text
- Replace images
- Adjust colors and fonts
- Rearrange sections
- Add new components

**4. Publish Instantly**
With one click, your website goes live on the internet, ready to share with the world.

### Common Questions

**Do I need coding experience?**
No! OpsVanta is designed for everyone. The AI handles the technical work.

**Can I use my own domain?**
Yes, you can connect any custom domain. We'll handle DNS and SSL setup.

**How much does it cost?**
We have a free plan to get started, with paid plans starting at $12/month.

**Can I export my website?**
Yes, you can export the code to host anywhere.

**Is my website SEO-friendly?**
Yes, all websites are optimized for search engines.

**Can I sell products?**
Yes, with e-commerce features available on higher plans.

**What if I need help?**
We offer email support, live chat, video tutorials, and documentation.
